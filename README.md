# Overview

A simple skeleton for spinning up small Ruby apps and Sinatra servers.

# Running

To run the main script

```bash
ruby main.rb
```

or

```bash
bundle exec rake main
```

To run specs

```bash
bundle exec rspec
```

or

```bash
bundle exec rake test
```

To run a Sinatra server

```bash
ruby server.rb
```

or

```bash
bundle exec rake server
```
